# Management System

## Prerequisites

- Docker
- `docker-compose`

## Run Server


- Terminal 1

   ```bash
   make up
   ```

- Terminal 2

   ```bash
   make run-backend
   ```

## Common Commands

### Init Data

```bash
make init-data
```

### Django Shell Plus

```bash
make shell
```

### Run Migrations

```bash
make migrations
make migrate
```
